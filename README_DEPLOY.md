# Publicar o Nota1000 ENEM na Vercel

Este projeto está pronto para deploy estático na Vercel.

## Configuração na Vercel

- Framework Preset: Other
- Build Command: deixe vazio
- Output Directory: deixe vazio ou use `.`
- Install Command: deixe vazio

## Fluxo recomendado

1. Crie um repositório no GitHub.
2. Envie esta pasta para o repositório.
3. Na Vercel, clique em Add New Project.
4. Importe o repositório.
5. Publique com as configurações acima.

Os arquivos `conexao.php` e `database.sql` são opcionais para uso local no XAMPP e não são enviados para a Vercel.
